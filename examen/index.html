Segunda parte:

Crear un formulario de registro para una tienda online

Diseña un formulario de registro para una tienda online que permita a los usuarios crear una cuenta. El formulario debe recopilar información personal, información de la cuenta y detalles de envío. Asegúrate de incluir campos obligatorios.

Requerimientos

Información Personal:
  • Nombre completo (obligatorio)
  • Correo electrónico (obligatorio)
  • Teléfono
  • Género (obligatorio, con opciones: Masculino, Femenino, Otro)
Información de la cuenta:
  • Nombre de usuario (obligatorio)
  • Contraseña (obligatorio)
  • Confirmar Contraseña (obligatorio)
Dirección de Envío:
  • Dirección Ciudad (obligatorio, con opciones: [lista de ciudades])
  • Código Postal
  • País (obligatorio, con opciones: [lista de países])
Casilla de verificación:
  • Crear un casilla de verificación con el texto: Acepto todas la condiciones.
Instrucciones:
  • Utiliza HTML para crear la estructura del formulario.
  • Utiliza CSS para aplicar estilos al formulario y hacerlo visualmente atractivo.
  • Los campos obligatorios deben marcarse como tales.
  • El formulario debe tener un botón de envío para enviar los datos.
